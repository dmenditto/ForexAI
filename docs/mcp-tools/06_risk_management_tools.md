# Risk Management MCP Tools

**Server**: Risk Management Server  
**Lambda**: `forex-risk-management`  
**Purpose**: Position sizing, risk metrics, trade validation

## Tools

1. **calculate_position_size** - Optimal position size based on risk %
2. **calculate_var** - Value at Risk (95% confidence)
3. **check_exposure** - Current currency exposure
4. **validate_trade** - Validate trade against risk parameters

**Used By**: Risk Management Agent
