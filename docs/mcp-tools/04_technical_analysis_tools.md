# Technical Analysis MCP Tools

**Server**: Technical Analysis Server  
**Lambda**: `forex-technical-analysis`  
**Purpose**: Calculate indicators and detect patterns  
**Implementation**: TA-Lib or Pandas-TA

## Tools

1. **calculate_indicator** - Calculate any technical indicator (RSI, MACD, SMA, EMA, BB, ATR)
2. **detect_patterns** - Identify chart patterns
3. **find_support_resistance** - Key support/resistance levels
4. **calculate_multiple_indicators** - Multiple indicators at once

**Used By**: Technical Analysis Agent
