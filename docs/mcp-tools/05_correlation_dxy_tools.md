# Correlation & DXY MCP Tools

**Server**: Correlation & DXY Server  
**Lambda**: `forex-correlation-dxy`  
**Purpose**: Currency correlations, DXY, JPY strength analysis  
**Critical For**: Forex-specific validation

## Tools

1. **get_dxy_data** - US Dollar Index current and historical
2. **get_currency_strength** - Individual currency strength index (0-100)
3. **calculate_correlation** - Correlation coefficient between pairs
4. **get_jpy_crosses** - All JPY pair data for risk sentiment
5. **validate_confluence** - Cross-pair confluence validation

**Used By**: Correlation & Confluence Agent
