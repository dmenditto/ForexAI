# News & Sentiment MCP Tools

**Server**: News & Sentiment Server  
**Lambda**: `forex-news-sentiment`  
**Purpose**: News aggregation, sentiment analysis, institutional positioning  
**Data Source**: NewsAPI, CFTC COT reports, Twitter API (optional)

## Tools

1. **get_news** - Fetch news articles with sentiment
2. **analyze_sentiment** - NLP sentiment analysis on text
3. **get_social_sentiment** - Social media sentiment
4. **get_cot_report** - Commitment of Traders positioning

**Used By**: Sentiment Analysis Agent
